# HireMate

HireMate is a full-stack AI-powered job application tracker for students and job seekers.

Users can generate tailored cover letters and manage job applications in a modern dashboard.

## Tech Stack
React, Node.js, Express, MongoDB Atlas, OpenRouter AI

## Features
- AI cover letter generation
- Save applications
- Dashboard tracking
- Delete applications
- Clean SaaS UI

## Run locally

Backend:
cd server
npm install
npm run dev

Frontend:
cd client
npm install
npm run dev
